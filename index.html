Como funciona a plataforma
1
7 níveis progressivos
Cada nível apresenta um conceito SQL com explicação, exemplos e exercícios práticos — do SELECT básico até LEFT JOIN. Um exercício intermediário de JOIN facilita a transição entre os níveis.
2
Editor com destaque de sintaxe
O editor colore automaticamente as palavras-chave SQL (SELECT, FROM, WHERE...), strings e números enquanto você digita. Use Ctrl+Enter para executar rapidamente.
3
Validação real do resultado
O sistema executa sua query e compara o resultado com a solução oficial — verificando colunas, linhas e valores. Erros de sintaxe não contam como tentativa incorreta.
4
Duas dicas por exercício
Dica 1 orienta o conceito sem revelar a resposta. Ver solução carrega a query correta no editor para você estudar e executar.
5
Pontuação e progresso salvos
Sem dicas: 15 pts. Com dica 1: 10 pts. Com solução: 5 pts. O progresso é salvo automaticamente no navegador — você pode fechar e retomar de onde parou.
6
Relatório de aprendizagem
Ao concluir todos os 26 exercícios, um relatório mostra: progresso por nível, tempo gasto, conceitos dominados, comandos SQL aprendidos e exercícios com mais dificuldade.

Como usar a plataforma
Navegação
Níveis e exercícios concluídos
  • Níveis com círculo verde na barra lateral são clicáveis — clique para rever a introdução com a mensagem do CEO e o conceito SQL
  • Os círculos numerados em Exercícios também são clicáveis após resolvidos — clique para voltar a qualquer exercício anterior
Editor SQL
Atalhos e botões
  • Ctrl + Enter — executa a query sem clicar no botão Executar
  • Limpar — apaga o editor e o resultado da última execução
  • Tentar novamente — esconde o banner de acerto e permite reescrever do zero, mesmo após ter resolvido o exercício
  • O editor destaca palavras-chave SQL em azul, strings em verde e números em amarelo automaticamente
Dicas
Como funcionam
  • Dica 1 — orientação conceitual, explica o caminho sem revelar a resposta
  • Ver solução — carrega a query correta no editor; vale 5 pts ao acertar
  • Após resolver o exercício, você pode rever as dicas sem penalidade de pontos
Banco de Dados
Abas na barra lateral
  • Tabelas — campos de cada tabela com indicação de PK e FK; clique no nome para expandir
  • Conexões — clique em "Ver diagrama completo" para abrir o diagrama em tela cheia com nós arrastáveis
Modo livre
Exploração sem exercício
  • Acesse pelo botão no topo da barra lateral a qualquer momento durante a jornada
  • Escreva qualquer consulta SELECT no banco da DataShop sem contexto de exercício
  • Útil para explorar os dados, testar queries ou praticar livremente
  • Apenas SELECT é permitido — INSERT, UPDATE, DELETE e outros comandos são bloqueados
Pontuação
Como os pontos são calculados
  • Sem dicas — 15 pts
  • Com a Dica 1 — 10 pts
  • Com a solução completa — 5 pts
  • Exercícios já resolvidos não somam pontos ao revisitar
Relatório de Aprendizagem
O que é mostrado
  • Aparece ao concluir todos os 26 exercícios
  • Barra de progresso geral com percentual de conclusão
  • Pontos acumulados, exercícios resolvidos sem dica, respostas incorretas e tempo total na plataforma
  • Progresso por nível e análise de domínio por conceito SQL
  • Lista dos exercícios com mais tentativas incorretas
Como respostas incorretas são contadas
  • Conta apenas queries que executaram com sucesso mas retornaram dados diferentes do resultado esperado
  • Erros de sintaxe SQL — palavra digitada errada, coluna inexistente, parêntese faltando — não são contados
  • Revisitar um exercício já resolvido e executar novas queries também não conta
